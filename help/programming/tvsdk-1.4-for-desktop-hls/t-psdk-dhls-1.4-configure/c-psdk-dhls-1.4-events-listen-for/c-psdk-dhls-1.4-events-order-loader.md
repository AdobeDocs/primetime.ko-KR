# 로더 이벤트{#loader-events}

플레이어는 다음 이벤트를 기반으로 작업을 구현할 수 있습니다.

| 이벤트 | 의미 |
|---|---|
| `MediaPlayerItemLoader.completed` | 미디어 리소스 로드가 완료되었습니다. |
| `MediaPlayerItemLoader.failed` | 미디어 리소스를 로드하는 도중 문제가 발생했습니다. |

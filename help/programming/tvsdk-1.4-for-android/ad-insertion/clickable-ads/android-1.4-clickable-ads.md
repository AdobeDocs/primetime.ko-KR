# 개요 {#clickable-ads-overview}

TVSDK는 클릭스루 광고에 대해 작업을 수행할 수 있도록 정보를 제공합니다. 플레이어 UI를 만들 때, 사용자가 클릭 가능한 광고를 클릭할 때 응답하는 방법을 결정해야 합니다.

Android용 TVSDK의 경우 선형 광고만 클릭할 수 있습니다.

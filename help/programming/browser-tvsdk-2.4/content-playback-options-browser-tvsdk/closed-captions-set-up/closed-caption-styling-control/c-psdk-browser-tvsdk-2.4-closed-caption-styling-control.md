# 개요 {#control-closed-caption-styling-overview}

TextFormat 클래스를 사용하여 자막 트랙의 스타일 정보를 제공할 수 있습니다. 플레이어에서 표시하는 모든 폐쇄 캡션의 스타일을 설정합니다.

이 클래스는 글꼴 유형, 크기, 색상 및 배경 불투명도와 같은 자막 스타일 정보를 캡슐화합니다.

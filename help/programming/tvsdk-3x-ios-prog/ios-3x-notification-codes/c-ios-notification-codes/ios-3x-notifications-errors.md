# 오류 알림 코드 {#error-notification-codes}

이 표에서는 ERROR 유형 알림에 대한 자세한 정보를 제공합니다.

대부분의 오류에는 관련 메타데이터가 포함됩니다(예: 다운로드하지 못한 리소스의 URL). 일부 알림에는 주 비디오 컨텐츠, 대체 오디오 컨텐츠 또는 광고에서 문제가 발생했는지 여부를 지정하는 메타데이터가 포함되어 있습니다.

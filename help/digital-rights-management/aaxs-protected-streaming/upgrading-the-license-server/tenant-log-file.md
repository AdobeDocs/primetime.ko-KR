# 테넌트 로그 파일{#tenant-log-file}

각 테넌트의 테넌트 로그 파일 [!DNL flashaccess-tenant.log]은 *LicenseServer.LogRoot* [!DNL /flashaccesserver/tenants/]*tenantname*&#x200B;에 있습니다. 테넌트 로그에는 이 테넌트에 대해 생성된 각 라이선스를 설명하는 감사 정보가 포함되어 있습니다.

# 전역 로그 파일{#global-log-file}

전역 로그 파일인 flashaccess-global.log는 LicenseServer.LogRoot *에 있습니다*. 이 로그에는 Adobe Access SDK에서 생성된 로그 메시지나 서버 초기화 동안 생성된 로그 메시지가 포함될 수 있습니다.

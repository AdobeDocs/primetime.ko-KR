# 로그 파일{#log-files}

보호된 스트리밍 응용 프로그램의 Adobe Access Server에서 생성된 로그 파일은 *LicenseServer.LogRoot에서 지정한 디렉토리에 있습니다*.

>[!NOTE]
>
>서버가 실행되는 동안 현재 로그 파일을 삭제하거나 이동하면 로그 파일이 다시 생성되지 않을 수 있으며 일부 로그 정보가 손실됩니다.

# 로그 파일{#log-files}

Adobe Access Server for Protected Streaming 응용 프로그램에서 생성한 로그 파일은에 지정된 디렉터리에 있습니다. *LicenseServer.Logroot*.

>[!NOTE]
>
>서버가 실행되는 동안 현재 로그 파일을 삭제하거나 이동하면 로그 파일이 다시 생성되지 않을 수 있으며 일부 로그 정보가 손실됩니다.

# 크로스 도메인 정책 파일 {#crossdomain-policy-file}

Flash Runtime 클라이언트가 라이센스 서버로부터 라이센스를 요청하려면 크로스 도메인 정책 파일이 필요합니다. 자세한 *내용은 Adobe Access SDK를 사용하여* 콘텐츠 보호를 참조하십시오.

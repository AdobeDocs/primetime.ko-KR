# 요구 사항{#requirements}

* Microsoft Windows Server 2008 또는 Red Hat® Enterprise Linux® 5.6
* Oracle Java JRE 1.6(사용자 지정 권한 부여 확장을 만들려면 Oracle Java JDK 1.6 필요)
* Apache Tomcat® 6(DVD의 타사\Tomcat\6.0.18 폴더에서 사용 가능)
* Adobe에서 발급한 자격 증명

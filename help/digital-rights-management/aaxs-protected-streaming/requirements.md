# 요구 사항{#requirements}

* Microsoft Windows Server 2008 또는 Red Hat® Enterprise Linux® 5.6
* Oracle Java JRE 1.6(사용자 정의 인증 익스텐션을 만들려면 Oracle Java JDK 1.6 필요)
* Apache Tomcat® 6(DVD의 세 번째 Party\Tomcat\6.0.18 폴더에서 사용 가능)
* Adobe에서 발행한 자격 증명

# Media Packager 사용 {#using-media-packager}

Media Packager를 사용하여 암호화할 파일의 데이터와 내용 파일에 적용할 정책을 지정할 수 있습니다. 예를 들어 비디오 데이터가 암호화되지만 오디오 데이터는 암호화되지 않도록 지정할 수 있습니다.

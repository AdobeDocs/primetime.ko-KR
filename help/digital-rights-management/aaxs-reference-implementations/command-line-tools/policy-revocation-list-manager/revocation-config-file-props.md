# 구성 파일 속성 {#configuration-file-properties}

다음은 해지 목록(라이센스 서버 인증서)에 대한 자격 증명을 포함하는 PKCS12 파일을 지정하는 해지 목록 관리자 속성입니다.

* `revocation.sign.certfile=license-server-credentials.pfx`
* `revocation.sign.certpass=password`

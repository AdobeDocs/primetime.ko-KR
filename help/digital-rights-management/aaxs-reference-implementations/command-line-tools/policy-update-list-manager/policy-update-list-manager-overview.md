# 정책 업데이트 목록 관리자 개요 {#policy-update-list-manager-overview}

정책 업데이트 목록 관리자를 사용하기 전에 요구 사항에 나열된 요구 사항을 충족하고 구성 파일에 필요한 정보가 포함되어 있는지 확인하십시오(*Adobe 액세스 참조 구현 사용*&#x200B;의 구성 파일 참조).

# 요구 사항 {#requirements}

다음을 설치했는지 확인해야 합니다.

* Tomcat 6.0 이상
* MySQL과 같은 데이터베이스(라이선스 서버 구성 요소에만 필요)
* Java 1.6 이상
* Ant 1.8 이상(샘플 빌드 스크립트 사용)

Tomcat 및 MySQL을 설치한 후 Adobe에서 자격 증명을 가져옵니다.

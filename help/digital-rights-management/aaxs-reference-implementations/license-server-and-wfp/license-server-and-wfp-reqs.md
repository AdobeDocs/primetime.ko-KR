# 요구 사항 {#requirements}

다음을 설치해야 합니다.

* Tomcat 6.0 이상
* MySQL과 같은 데이터베이스(라이센스 서버 구성 요소에 대해서만 필요)
* Java 1.6 이상
* Ant 1.8 이상(샘플 빌드 스크립트 사용)

Tomcat 및 MySQL을 설치한 후에는 Adobe에서 자격 증명을 얻습니다.

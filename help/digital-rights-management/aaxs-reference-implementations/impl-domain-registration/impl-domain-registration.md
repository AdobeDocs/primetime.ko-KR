# 도메인 등록 구현 {#implementing-domain-registration}

참조 구현 라이선스 서버는 도메인 등록 비즈니스 로직의 두 가지 변형을 보여 줍니다. 일반적인 도메인 서버는 다음 워크플로우 중 하나만 구현합니다.

* ID 기반 도메인
* 익명 도메인

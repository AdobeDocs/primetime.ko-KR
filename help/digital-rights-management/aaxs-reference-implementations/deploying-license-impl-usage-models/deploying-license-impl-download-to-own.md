# 소유 다운로드 {#download-to-own}

DTO(Download-To-Own) 사용 모델을 사용하면 사용자는 온라인 또는 오프라인으로 사용할 콘텐츠를 다운로드할 수 있으며 콘텐츠에 대한 영구 라이선스를 발급받습니다. 라이선스를 요청할 때 사용자가 콘텐츠를 구매했는지 확인할 수 있도록 사용자를 인증해야 합니다.

# 상대적 종료일{#relative-end-date}

상대적 종료 날짜는 라이선스가 발급된 날짜와는 관련이 없고 포장 날짜를 기준으로 표시되는 라이선스 만료 날짜를 지정합니다.

예제 사용 사례: 자동화된 패키징 프로세스에서 일련의 비디오에 대해 이 옵션과 함께 단일 Primetime DRM 정책을 사용하여 만료 날짜를 패키징 날짜를 기준으로 30일로 설정합니다.

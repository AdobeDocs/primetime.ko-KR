# 도메인 간 정책 파일{#cross-domain-policy-file}

Flash 런타임 클라이언트가 라이센스 서버에서 라이센스를 요청해야 하는 경우 크로스 도메인 DRM 정책 파일이 필요합니다. 크로스 도메인 정책에 대한 자세한 내용은 *콘텐츠 보호를 위한 Adobe Primetime DRM SDK 사용을 참조하십시오*.

# 정책 관리자 명령줄 사용 {#policy-manager-command-line-usage}

```
java -jar AdobePolicyManager.jar  
<i class="+ topic ph hi-d="" i "="">
  command filename [options] 
</i class="+ topic>
```

**표 1:명령**

| 명령 | 설명 |
|---|---|
| `new` | 새 DRM 정책 만들기 |
| `detail` | 기존 DRM 정책에 대해 설명합니다. |
| `update` | 기존 DRM 정책 업데이트 |

# SSL {#configuring-ssl} 구성

SSL(Secure Sockets Layer)은 Flash Media Rights Management Server 1.x 클라이언트에 대한 지원이 필요한 경우에만 필요합니다. Adobe 액세스 키 서버에 클라이언트 인증이 있는 SSL이 필요합니다. 자세한 내용은 *Adobe 액세스 키 서버 사용을 참조하십시오.

# 무작위 숫자 생성{#generating-random-numbers}

하드웨어 무작위 번호 생성기는 충분한 엔트로피가 생성되도록 Linux 서버에서 사용할 수 있습니다. 시스템에서 충분한 엔트로피를 생성할 수 없는 경우 랜덤의 소스가 필요한 Adobe Access 작업은 데이터를 기다리는 동안 차단됩니다 `/dev/random`.

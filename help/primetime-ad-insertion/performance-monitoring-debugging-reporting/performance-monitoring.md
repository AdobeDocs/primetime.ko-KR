# 성능 모니터링 {#monitoring-performance}

Primetime Ad Insertion은 [모니터링 및 디버깅 도구](https://ssai.console.primetime.adobe.com/) 광고 삽입 성능 및 서비스 상태(예: http 성공/오류율, 동시 요청/세션, 광고 크리에이티브 통계, 디버깅 등)를 모니터링하기 위한 대시보드를 포함합니다.

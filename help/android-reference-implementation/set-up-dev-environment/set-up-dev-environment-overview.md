# 개요 {#overview}

TVSDK Primetime 참조 Android 프로젝트는 주요 구성이나 IDE 변경 사항 없이 Eclipse와 호환되는 방식으로 구성됩니다. Primetime 참조 프로젝트를 설정하려면 먼저 지침에 따라 Eclipse를 구성한 다음 프로젝트를 빌드하는 단계를 따릅니다.

# 개요 {#overview}

TVSDK Primetime Reference Android 프로젝트는 Eclipse와 호환되는 방식으로 구성되며 주요 구성이나 IDE가 변경되지 않습니다. Primetime 참조 프로젝트를 설정하려면 먼저 지침에 따라 Eclipse를 구성한 다음 해당 단계에 따라 프로젝트를 제작하십시오.

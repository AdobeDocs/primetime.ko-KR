# 기능 관리자를 사용하는 방법 {#how-to-use-feature-managers}

기능 관리자를 사용하여 기능을 활성화, 비활성화 및 구성하는 방법에는 두 가지가 있습니다.

* 구성 파일의 구성 정보를 미디어 플레이어에 전달하여 기능 관리자를 만들고 구성할 수 있습니다.
* 관리자 팩터리를 사용하여 코드를 통하지 않고 기능을 활성화 및 비활성화할 수 있습니다.

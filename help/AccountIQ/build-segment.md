# 계정 IQ에서 세그먼트 작성 {#build-segment}

분석을 위해 새 세그먼트를 작성하려면 다음 작업을 수행하십시오.

1. 다음 항목 선택 **새 세그먼트 만들기** 옵션 켜기 **세그먼트** 페이지를 가리키도록 업데이트하는 중입니다.
1. 에 세그먼트 이름 추가 **세그먼트 이름**.
